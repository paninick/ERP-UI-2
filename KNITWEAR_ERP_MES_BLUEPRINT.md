# 针织服装工贸一体 ERP / MES 业务蓝图

最后更新：2026-04-24

## 1. 目标定位

本蓝图适用于以下类型企业：

- 工贸一体化针织服装企业
- 集团总部 + 多工厂 / 多分厂 / 多车间模式
- 毛衫、拼接款、普通针织衫并行经营
- 既有全工序总厂，也有按工序拆分的分厂
- 存在外协加工厂与第三方检品公司协同
- 以日系客户和日单质量要求为主要业务特征

规划原则：

- 业务链优先，不为界面方便牺牲真实流程
- 一次录入，多环节继承，禁止重复造数
- 产品身份唯一，部门编号分层，不混淆用途
- 计划、生产、质检、成本四账分离但链路贯通
- 打印、扫码、报工必须服务车间执行

## 2. 总体业务判断

建议采用“集团贸易中台 + 技术中台 + 多工厂执行单元 + 外协 / 检品协同”的架构。

核心判断如下：

- `styleCode` 是产品身份，不应被计划号、工单号替代
- 销售下单时必须锁定客户、款号、数量、交期、质量要求
- 技术、计划、采购、生产、质检分别使用自己的控制编号，但都必须继承上游源头
- 下游单据不重新维护款号、数量、交期，只允许补充本部门业务字段
- 生产执行必须按工序、工作中心、员工、班组、工厂维度沉淀真实数据
- 日单必须增加检品公司预约、检品结果、出货放行控制

## 3. 企业组织模型

建议组织层级如下：

1. 集团 / 总公司
2. 事业部 / 品牌线
3. 工厂
4. 分厂 / 车间
5. 班组
6. 工位 / 设备

建议组织类型字典：

- `HEAD_OFFICE` 总部
- `BUSINESS_UNIT` 事业部
- `FACTORY` 工厂
- `BRANCH_FACTORY` 分厂
- `WORKSHOP` 车间
- `TEAM` 班组
- `STATION` 工位
- `OUTSOURCE_VENDOR` 外协厂
- `INSPECTION_COMPANY` 检品公司
- `WAREHOUSE_UNIT` 仓储单位

推荐工厂建模方式：

- 总公司：负责销售、技术、打样、PMC、采购、财务、单证、质量标准
- 毛衫厂：横机、套口、水洗、检验、后整、包装
- 服装厂：裁剪、缝制、整烫、检验、包装
- 混合厂：同时承接毛衫和缝制工艺
- 专业分厂：横机分厂、套口分厂、水洗分厂、包装分厂

## 4. 产品与工艺分类

### 4.1 毛衫

- 横机织片
- 套口

- 检验
- 后整与包装

### 4.2 拼接款

主工艺特征：

- 横机织片
- 套口或局部成型
- 缝制拼接
- 水洗 / 定型
- 检验
- 后整与包装

### 4.3 普通针织衫

主工艺特征：

- 面辅料准备
- 裁剪 / 缝制
- 整烫
- 检验
- 检针
- 包装

每个产品族都应有：

- 款号主档
- 工艺路线模板
- BOM 版本
- 质量标准模板
- 包装模板
- 打印模板

## 7. 单据与编号体系

建议保留多编号，但明确职能：

- `styleCode`：产品身份号
- `customerStyleCode`：客户款号
- `salesNo`：销售订单号
- `techNo` / `noticeNo`：工艺通知号 / 技术版本号
- `sampleNo`：打样号
- `bomVersion`：BOM 版本号
- `routeVersion`：工艺路线版本号
- `planNo`：计划单号
- `purchaseNo`：采购单号
- `outsourceNo`：外协单号
- `jobNo`：生产单号
- `processCardNo`：工序卡号
- `inspectionNo`：检验单号
- `shipmentNo`：出货单号

编号治理原则：

- 一个编号只表达一种业务责任
- 下游单据必须回链上游单号
- 打印件同时展示“当前单号 + 上游源头号 + 款号”
- 车间扫码优先跳执行页面，不跳纯展示页

推荐继承关系：

`styleCode -> salesNo -> techNo -> planNo -> jobNo -> processCardNo -> inspectionNo -> shipmentNo`

## 9. 三条标准工艺主线

### 9.1 毛衫标准路线

建议流程：

销售订单
-> 技术确认
-> 打样
-> BOM 发布
-> 纱线 / 辅料采购
-> 横机织片
-> 套口
-> 水洗 / 缩绒 / 定型
-> 初检
-> 锁扣 / 钉标 / 整烫
-> 吊牌
-> 检针
-> 包装
-> 成品检验
-> 入库
-> 出货

关键控制点：

- 织片数量与片数
- 套口收发数量
- 水洗前后损耗
- 缩率 / 手感 / 尺寸稳定性
- 残针与检针控制
- 包装配套完整性

### 9.2 拼接款标准路线

建议流程：

销售订单
-> 技术确认
-> 样衣确认
-> BOM / 工艺发布
-> 纱线与布料采购
-> 横机织片
-> 套口或局部成型
-> 缝制拼接
-> 水洗 / 定型
-> 中查
-> 整烫
-> 吊牌 / 辅料安装
-> 检针
-> 包装
-> OQC
-> 出货

关键控制点：

- 拼接部位一致性
- 织片与布片配套
- 尺寸匹配
- 色差
- 洗后变形

### 9.3 普通针织衫标准路线

建议流程：

销售订单
-> 工艺确认
-> 面辅料采购
-> 裁剪
-> 缝制
-> 整烫
-> 检验
-> 检针
-> 包装
-> 入库
-> 出货

关键控制点：

- 裁片配套
- 缝制不良
- 尺寸允差
- 外观疵点
- 检针

## 10. 计划、实际、质量、成本的权责边界

### 10.1 计划权限

计划部门负责：

- 接单产能评估
- 工厂分配
- 交期倒排
- 主计划与周计划
- 变更协调

计划部门不得直接修改：

- 销售承诺数量
- 技术版本内容
- 质检判定结果
- 财务单价

### 10.2 生产权限

生产部门负责：

- 生产单执行
- 工序报工
- 转工序
- 班组排班
- 现场异常反馈

生产部门不得直接修改：

- 销售订单源数据
- 质量放行结论
- 标准 BOM
- 客户价格

### 10.3 品质权限

品质部门负责：

- 来料检验
- 制程检验
- 终检
- 缺陷判定
- 放行 / 驳回
- 检品公司对接

品质部门不得直接修改：

- 生产实绩数量
- 工艺基础版本
- 财务成本

### 10.4 成本权限

财务 / 成本部门负责：

- 材料成本
- 工序人工
- 外协成本
- 次品返工损耗成本
- 结算与利润分析

成本数据建议仅开放给：

- 财务
- 成本会计
- 总经理 / 授权高管

## 12. 日单与日系客户特殊控制

日单不应只用备注控制，建议形成“客户质量画像”。

建议客户档案增加字段：

- 是否日单客户
- 是否必须第三方检品
- AQL 等级
- 尺寸允差标准
- 检针要求
- 包装折叠要求
- 吊牌 / 洗标 / 唛头规则
- 箱唛规则
- 出货前检品提前期
- 未放行是否禁止出运

建议新增业务节点：

`OQC -> 检品公司预约 -> 检品结果 -> 出货放行`

日单必须沉淀的记录：

- 尺寸实测
- 洗后测试
- 检针记录
- 缺陷照片
- 复检记录
- 检品公司报告号
- 最终放行人

## 13. 外协协同模型

外协类型建议区分：

- 整单外发
- 单工序外发
- 水洗外发
- 缝制外发
- 后整外发
- 检验外发

外协必须记录：

- 外协厂
- 外协工序
- 发出数量
- 收回数量
- 合格数量
- 不良数量
- 单价
- 结算方式
- 质检结果
- 责任归属

外协不能只作为采购单延伸，建议独立外协执行单。

- 物料成本

- 外协成本

- 次品
- 返工
- 补片
- 重洗
- 降级
- 报废

## 17. 当前项目可沿用的系统主线

根据当前前端项目已形成的能力，可以保留以下主线继续深化：

- 销售订单锁定源头
- 计划单继承销售订单
- 生产单继承计划单
- 工序报工形成真实流转
- 质检放行控制后续出货
- 打印与扫码回流到业务页面

不建议推翻重来，应继续补齐：

- 组织与岗位字典
- 工厂 / 车间 / 班组维度
- 工艺模板
- 日单客户质量画像
- 检品公司节点
- 外协执行与结算闭环

## 18. 分阶段落地建议

### 第一阶段：主数据收口

- 统一款号、客户款号、订单号、计划号、生产号含义
- 恢复并补齐组织、岗位、工种、缺陷、检验字典
- 固化客户画像、产品族、工艺模板

### 第二阶段：执行链打通

- 销售 -> 技术 -> 计划 -> 生产 -> 质检 -> 出货全链只做继承
- 建立分厂、车间、班组、工位执行维度
- 按产品族上线标准工艺路线模板

### 第三阶段：质量与日单控制

- 补齐 IQC / IPQC / FQC / OQC
- 增加检针、尺寸实测、洗后测试
- 增加第三方检品公司预约与放行节点

### 第四阶段：成本与经营分析

- 物料、人工、外协、质量损失成本联动
- 形成订单毛利、款号毛利、工厂绩效、车间效率分析

### 第五阶段：管理驾驶舱

- 订单进度
- 款号进度
- 工厂产能
- 异常闭环
- 日单放行状态
- 成本偏差分析

## 19. 推荐优先级

最优先，不可再推迟：

1. 组织 / 岗位 / 工种 / 班组模型
2. 客户质量画像与日单控制
3. 三类产品工艺模板
4. 外协与检品公司协同
5. 质量与成本留痕

## 20. 可参考的标准类别

以下内容适合作为制度与字典设计的参照方向：

- GB/T 2828.1 抽样检验与 AQL 体系
- GB 18401 纺织产品安全技术规范
- GB/T 5296.4 纺织品和服装使用说明
- GB/T 1335 系列服装号型标准
- FZ/T 80003 缝纫型式分类和术语
- FZ/T 70018 针织服装理化性能要求
- 国家职业技能标准中的服装制版师、缝纫工等岗位标准

本蓝图采用的是“行业通行做法 + 标准治理框架 + 当前系统既有数据链”的综合规划思路，不替代企业内部管理制度与客户个别协议。
